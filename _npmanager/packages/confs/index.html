(!) Let you know how to start/stop/reload daemons.

# Start NGINX
$ sudo npmanager start nginx
# Stop NGINX
$ sudo npmanager stop nginx
# Reload NGINX
$ sudo npmanager reload nginx
# Restart NGINX
$ sudo npmanager restart nginx

# Start PHP
$ sudo npmanager start php
# Stop PHP
$ sudo npmanager stop php
# Reload PHP
$ sudo npmanager reload php
# Restart PHP
$ sudo npmanager restart php

# Start MariaDB
$ sudo npmanager start mariadb
# Stop MariaDB
$ sudo npmanager stop mariadb
# Reload MariaDB
$ sudo npmanager reload mariadb
# Restart MariaDB
$ sudo npmanager restart mariadb

(!) Where are configuration files located?

# NGINX
$ cd /etc/nginx
$ vim nginx.conf
# NGINX Web
$ cd /etc/nginx/conf.d
$ vim localhost
# PHP
$ cd /etc/php5/fpm
$ vim php.ini
# MariaDB
$ cd /etc/mysql
$ vim my.cnf

(!) How to upgrade daemons when there's a new version released?

$ sudo apt-get update
$ sudo apt-get upgrade
